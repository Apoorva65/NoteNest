# NoteNest

NoteNest is an offline-first notes application built with React.

## Features

- Create, edit, and delete notes
- Auto-save while typing
- Offline persistence using LocalStorage
- Sidebar navigation with active note highlight
- Search notes by title or content
- Notes sorted by last edited time
- Last edited timestamp display

## Routing

- `/` – Empty editor state
- `/note/:id` – View or edit a note
- `/new` – Create a new note (redirects automatically)

## Tech Stack

- React (Vite)
- React Router
- Tailwind CSS
- LocalStorage
